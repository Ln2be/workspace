حل المعادلات باستخدام الاستبدال

التمرين النموذجي

حل المعادلة: \( e^{2x} - 4e^x + 3 = 0 \)

الخطوة 1: الاستبدال

نضع \( X = e^x \) فتتحول المعادلة إلى:

\[ X^2 - 4X + 3 = 0 \]

الخطوة 2: حل المعادلة التربيعية

\[ (X - 1)(X - 3) = 0 \]

الحلول: \( X = 1 \) أو \( X = 3 \)

الخطوة 3: الرجوع للمتغير الأصلي

\[ e^x = 1 \Rightarrow x = 0 \]

\[ e^x = 3 \Rightarrow x = \ln 3 \]

تمارين تطبيقية

التمرين 1

\( (\ln x)^2 + 2\ln x - 3 = 0 \)

الاستبدال: \( X = \ln x \)

\[ X^2 + 2X - 3 = 0 \Rightarrow (X+3)(X-1)=0 \]

الحلول: \( X = -3 \) أو \( X = 1 \)

\[ x = e^{-3} \quad,\quad x = e \]

التمرين 2

\( 2e^{2x} - 5e^x + 2 = 0 \)

الاستبدال: \( X = e^x \)

\[ 2X^2 - 5X + 2 = 0 \]

\[ X = 2 \quad,\quad X = \frac{1}{2} \]

\[ x = \ln 2 \quad,\quad x = -\ln 2 \]
