Exercice 2 : Service au volley-ball

📝 Énoncé

Un joueur lance un ballon avec une vitesse initiale V0 faisant un angle α = 45° avec l'horizontale. Le ballon est frappé à une hauteur h = 0,50 m du sol et à une distance D = 9 m du filet. La hauteur du filet est H = 2,43 m. Le service est réussi si :

  • Le ballon passe au-dessus du filet (en x = D, z = H).
  • Il touche le sol dans le camp adverse entre le filet et la limite à x = 2D.
  1. Caractériser le vecteur accélération du ballon.
  2. Déterminer les équations horaires du vecteur vitesse V.
  3. Déterminer les équations horaires du vecteur position OG.
  4. Étudier la trajectoire :
    1. Équation de la trajectoire
    2. Nature de la trajectoire
    3. Type de mouvement
  5. Déterminer Vmin (passe juste au-dessus du filet) et Vmax (tombe à x = 2D).
  6. Calculer la distance entre le filet et le point de chute pour V0 = 12 m/s.

🔍 Solution Synthétique

1. Accélération

ax = 0 (mouvement horizontal uniforme)
az = -g = -10 m/s² (chute libre)

2. Équations horaires de la vitesse

Vx(t) = V0·cosα
Vz(t) = -g·t + V0·sinα

3. Équations horaires de la position

x(t) = V0·cosα·t
z(t) = -½ g t² + V0·sinα·t + h

4. Trajectoire

z(x) = -[g/(2V0²·cos²α)]·x² + tanα·x + h
Parabole (mouvement de chute libre).

5. Vitesses limites

Vmin : Solution de H = -[g·D²]/[2Vmin²·cos²α] + D·tanα + h
Vmin ≈ 10,6 m/s

Vmax : Solution de 0 = -[g·(2D)²]/[2Vmax²·cos²α] + 2D·tanα + h
Vmax ≈ 13,1 m/s

6. Distance de chute

Pour V0 = 12 m/s, résoudre z(x) = 0 :
x ≈ 15,16 m ⇒ Distance depuis le filet = 15,16 - 9 = 6,16 m
