Exercice I : Mouvement d'une Bille

1. Vecteurs vitesse et position :
• Accélération : \( \vec{a} = \frac{d\vec{v}}{dt} = -4\vec{j} \)
• Intégration vitesse : \( \vec{v}(t) = \int \vec{a} \, dt = 3\vec{i} + (-4t + 2)\vec{j} \)
• Intégration position : \( \overrightarrow{OM}(t) = \int \vec{v} \, dt = 3t\vec{i} + (-2t^2 + 2t)\vec{j} \)
\(\boxed{\vec{v}(t) = 3\vec{i} + (-4t + 2)\vec{j} \quad \overrightarrow{OM}(t) = 3t\vec{i} + (-2t^2 + 2t)\vec{j}}\)
2. Trajectoire :
• De \( x(t) = 3t \) → \( t = \frac{x}{3} \).
• Substitution dans \( y(t) \) : \( y = -2\left(\frac{x}{3}\right)^2 + 2\left(\frac{x}{3}\right) \)
\(\boxed{y = -\frac{2}{9}x^2 + \frac{2}{3}x}\)
3. Point S (altitude max) :
• Condition : \( \frac{dy}{dt} = -4t_S + 2 = 0 \) → \( t_S = 0.5 \, \text{s} \).
• Coordonnées : \( x_S = 1.5 \, \text{m}, y_S = 0.5 \, \text{m} \).
• Vitesse : \( \vec{v}_S = 3\vec{i} \, \text{m/s} \) (car \( v_y = 0 \))
\(\boxed{S(1.5\, \text{m}, 0.5\, \text{m}) \quad \vec{v}_S = 3\vec{i}\, \text{m/s}}\)
4. Point I (\( y = 0 \)) :
• Temps : \( -2t_I^2 + 2t_I = 0 \) → \( t_I = 1 \, \text{s} \).
• Vitesse : \( \vec{v}_I = 3\vec{i} - 2\vec{j} \).
• Angle \( \alpha \) : \( \tan\alpha = \frac{2}{3} \) → \( \alpha \approx 33.7^\circ \).
• Accélérations : \( a_t = \frac{d|\vec{v}|}{dt} = 2.22 \, \text{m/s}^2, \, a_n = \sqrt{a^2 - a_t^2} = 3.33 \, \text{m/s}^2 \).
• Rayon : \( R = \frac{|\vec{v}|^2}{a_n} = 3.9 \, \text{m} \).
• Accélérations: \( \cos(\theta) = \frac{a_t}{a}\)→ \( a_t = a\cos(\theta) \) on a \(\theta = 90 - \alpha\)
\(\boxed{\alpha = 33.7^\circ, \, a_t = 2.22\, \text{m/s}^2, \, a_n = 3.33\, \text{m/s}^2, \, R = 3.9\, \text{m}}\)

Exercice II : Mouvement du Plateau

1. Nature du mouvement :
• Distances : \( 2.5\, \text{m}, 3.75\, \text{m}, 5\, \text{m}, 6.25\, \text{m}, 7.5\, \text{m} \).
• Suite arithmétique → MRUV.
• Accélération : \( a = \frac{r}{\tau^2} = \frac{1.25}{0.5^2} = 5\, \text{m/s}^2 \)
\(\boxed{\text{MRUV avec } a = 5\, \text{m/s}^2}\)
2. Équation horaire :
• Formule : \( x_p(t) = \frac{1}{2}at^2 + V_0t + x_0 \).
• Substitution : \( x_p(t) = 2.5t^2 + V_0t - 7.5 \)
\(\boxed{x_p(t) = 2.5t^2 + V_0t - 7.5}\)
3. Vitesse initiale \( V_0 \) :
• Condition : \( x_p(1) = 3\, \text{m} \).
• Équation : \( 2.5(1)^2 + V_0(1) - 7.5 = 3 \) → \( V_0 = 8\, \text{m/s} \)
\(\boxed{V_0 = 8\, \text{m/s}}\)
