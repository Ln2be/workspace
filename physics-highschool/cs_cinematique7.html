Dynamique du Point Matériel

1. Référentiels

Solide de référence pour mouvement :

  • Terrestre : Axes fixes Terre
  • Géocentrique : Centre Terre + étoiles
  • Copernic : Centre système solaire

2. Cinématique

Position

\(\overrightarrow{OM}(t) = x(t)\vec{i} + y(t)\vec{j} + z(t)\vec{k}\)

Vitesse

\(\vec{v} = \frac{dx}{dt}\vec{i} + \frac{dy}{dt}\vec{j} + \frac{dz}{dt}\vec{k}\)

Accélération

\(\vec{a} = \frac{d^2\overrightarrow{OM}}{dt^2}\)

3. Mouvements

MRU

\(x(t) = v_0 t + x_0\)

MRUV

\(v(t) = a t + v_0\)
\(x(t) = \frac{1}{2}a t^2 + v_0 t + x_0\)

4. Expérimentale

  • Vitesse moyenne : \(v_m = \frac{A_{n-1}A_{n+1}}{2\theta}\)
  • Accélération : \(a_m = \frac{v_{n+1} - v_{n-1}}{2\theta}\)
  • MRUV : \(r = a \theta^2\) (suite arithmétique)
