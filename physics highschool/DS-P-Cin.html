Règles Essentielles en Physique (Lycée)

1. Cinématique du Point Matériel

Vecteurs Fondamentaux

Position : OM⃗ = x(t)i⃗ + y(t)j⃗ + z(t)k⃗

Vitesse : v⃗ = d(OM⃗)/dt = vₓi⃗ + vᵧj⃗ + vzk⃗

Accélération : a⃗ = dv⃗/dt = aₓi⃗ + aᵧj⃗ + azk⃗

2. Mouvements Types

MRU (Vitesse Constante)

Équation horaire : x(t) = v₀t + x₀

MRUV (Accélération Constante)

Vitesse : v(t) = a⋅t + v₀

Position : x(t) = ½a⋅t² + v₀t + x₀

Relation sans temps : v² - v₀² = 2a(x - x₀)

3. Dynamique (RFD)

ΣF⃗ext = m⋅a⃗

Étapes :

  1. Isoler le système
  2. Lister les forces extérieures
  3. Projeter sur les axes
  4. Résoudre pour trouver a⃗

4. Référentiels

  • 🌍 Terrestre : Mouvements locaux
  • 🛰️ Géocentrique : Satellites
  • 🌞 Copernic : Planètes

💡 Astuce pour les Exercices

Pour un dépassement de véhicules :

Δx = vrelative × t

Utilisez x(t) = ½a⋅t² + v₀t + x₀ pour les MRUV.
